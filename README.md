# 🀄️ Tamenchan Bootcamp (タメンチャン・ブートキャンプ)

麻雀の「多面張（タメンチャン）」を特訓するWebアプリです！
複雑な形も一瞬で待ちがわかるようになっちゃおう✨

## 🚀 Features

- **ランダム問題生成**: 毎回違う形が出るから無限に練習できるよ！
- **即時判定**: タップするだけで正解・不正解がすぐわかる！
- **ギャルFeedback**: ギャルエンジニアが優しく（たまに厳しく？）教えてくれるよ💖

## 🛠 Tech Stack

- React 19
- TypeScript
- Vite
- Vanilla CSS (No Framework, Pure Vibes!)
- Vitest

## 💻 Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm run test
```
